Markdown Blog ハリボテ
=======

[teloo/markdown-blog](https://github.com/teloo/markdown-blog) の本実装の前に画面イメージをつけるために作ったハリボテです。

## Installation

```shell
$ git clone git@github.com:teloo/markdown-blog-haribote.git
$ cd markdown-blog-haribote
$ npm install --no-save
```

## Usage

以下のコマンドを実行後、ブラウザで http://localhost:8080 へアクセスしてください。

```shell
$ npm start
```
